## Anonym Chat Room
Aplikasi ruang obrolan secara realtime menggunakan Nodejs dan SocketIO. 

Pada contoh aplikasi dibawah, saya menggunakan 2 browser untuk melakukan pengujian, Edge(kiri) & Brave(kanan).
